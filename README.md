# Noctra 

Noctra is a modern, responsive data exploration dashboard built with a focus on
performance, accessibility, and clean UI design.

*Features*
- Dark-first UI with pastel accents
- Responsive dashboard layout
- Mobile-friendly navigation
- API-driven data views
- Accessible & performance-conscious design

*Tech Stack*
- React (Vite)
- Tailwind 
- Vanilla JavaScript principles
- REST APIs

*Purpose*
This project is built as a portfolio-grade frontend application to demonstrate
modern UI architecture, responsive design, and real-world data handling.
